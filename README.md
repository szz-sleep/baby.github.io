鸿奇添彩友俱乐部 · 彩票推荐记事本 —— 软件介绍与使用说明
🎯 软件简介
这是一款专为彩友设计的号码推荐编辑与记录工具。软件采用 9:16 竖版A4比例设计，界面简洁友好。支持 3D 与 排列3 双模式一键切换，具备智能自适应字号、历史记录归档、个性化底色/底图、高清截图及打印等功能，是您日常记录和分析号码的得力助手。

🚀 核心功能与操作指南
1. 模式切换（3D / 排列3）
如何操作：点击左上角的 ☰（菜单） 按钮，在展开的下拉菜单中选择 “排列3” 或 “3D”。
效果：选择后，标题中间的图标、下方的期号标签以及正文框中的默认示例内容会自动同步更新。

2. 日期与期号编辑
如何操作：直接在界面上的 “日期” 或 “期号” 文字位置进行点击，即可进入编辑模式输入您想要的内容。
如何保存：输入完成后，按下回车键或点击页面空白处，系统会自动保存您修改的内容。

3. 正文记录与智能自适应字号
如何操作：在中间圆角矩形空白区域内，任意编写您的号码推荐、走势分析等内容。
核心亮点：正文区域具备智能字号自适应功能。当您输入的内容较多，接近填满框体时，字号会自动缩小以免溢出；当您删除内容，文本减少后，字号会自动放大，最终恢复至默认大小。无需手动调整，全程自动适配。

4. 历史记录（保存与调用）
保存记录：点击左上角 💾（保存） 按钮，在弹出的输入框中输入您的笔记标题（如“昨天推荐”）并确认，即可将当前正文内容保存。
查看与管理：点击左上角 ☰（菜单），选择 “历史记录”。
加载：在历史记录列表中点击 “加载”，系统会自动将对应记录的内容、日期与模式重置到主界面，方便您继续编辑。
删除：点击 “删除” 可移除不需要的记录（删除前会有确认提示）。

5. 个性化外观定制
修改底色：点击 ☰（菜单） → “底色”。色板会以弹窗形式居中显示，无论电脑还是手机都能清晰操作，选中颜色后自动生效并保存。
修改底图：点击 ☰（菜单） → “底图”，选择一张图片上传。
效果：上传的图片会作为正文区域的背景显示，透明度被强制锁定为 15%，既美观又不会影响文字阅读。
清除底图：双击下拉菜单中的 “底图” 按钮，即可一键清除背景图。

6. 截图与打印
截图：点击 ☰（菜单） → “截图”，系统会自动截取当前卡片主界面的内容，并保存为高清 PNG 图片（系统会自动处理图文颜色，确保截图清晰无长条底色块）。在手机上截图时，会优先调用系统的分享菜单，方便您直接保存到相册或分享给好友。
打印：点击 ☰（菜单） → “打印”，系统会调用浏览器的打印功能，方便您导出为 PDF 或纸质打印。

7. 一键恢复默认
如何操作：点击 ☰（菜单） → “默认”。
效果：软件将一键清空所有自定义设置（底色、底图、自定义日期/期号、保存的 LOGO），并将正文内容重置为系统初始状态。
